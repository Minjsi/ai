#1 파이썬 가상 환경 설정

파이썬 프로젝트를 진행할 때 독립된 환경을 만들어준다. 예를 들면 2개 또는 3개의 프로젝트를 개발하고 관찰 할 때 각각 독립적인 환경에서 프로젝트들을 진행해야할 경우가 있다.

cd .. 디렉토리 하나 위로 가기
cd / 한번에 열개 위로 올라가기
md venvs 디렉토리 만들기
python -m venv mysite 파이썬 가상환경 셋팅
mysite라는 이름의 가상환경 

cd mysite 이동 후 
Scripts 이동
activate 입력하면 가상환경 실행
deactivate 입력 > 가상환경 나오기

C:\venvs\mysite\Scripts>





# 가상환경에 Django 설치
(mysite) C:\venvs\mysite\Scripts>pip install django==3.1.3

# 장고의 프로젝트 생성
장고의 프로젝트 개념은 하나의 사이트라고 보면된다. 
장고 프로젝트를 생성하면 한개의 웹 사이트를 생성하는 것이다.
하나의 프로젝트 안에는 여러개의 앱을 만들게 된다.
장고에서의 앱은 사이트를 구성하는 특정 기능이 구현된 작업 단위를 의미한다.

cd /
 md projects
cd projects
md mysite
cd mysite

(mysite) C:\projects\mysite>
django-admin startproject config .

python manage.py runserver

-> http://127.0.0.1:8000/
로켓 뜨면 셋팅 끝

django 한글화(지역화)

config - settings.py

LANGUAGE_CODE = 'en-us'

TIME_ZONE = 'UTC'

->>
LANGUAGE_CODE = 'ko-kr'

TIME_ZONE = 'Asia/seoul'
